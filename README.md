# 🌿 Salalah RV Park

موقع حجز احترافي لـ Salalah RV Park مبني بـ Next.js

---

## طريقة الرفع على Vercel

### الخطوة 1 — حمّل Node.js
https://nodejs.org (اختر LTS)

### الخطوة 2 — فك ضغط المشروع
فك ضغط الملف salalah-rv-park.zip في أي مكان على جهازك

### الخطوة 3 — ارفع على GitHub
1. اذهب إلى https://github.com وسجّل دخول
2. اضغط New Repository
3. اسمه: salalah-rv-park
4. اضغط Create
5. اسحب مجلد المشروع وأفلته في الصفحة

### الخطوة 4 — انشر على Vercel
1. اذهب إلى https://vercel.com
2. سجّل دخول بحساب GitHub
3. اضغط "Add New Project"
4. اختر مستودع salalah-rv-park
5. اضغط Deploy
6. انتظر دقيقة واحدة ✅

### النتيجة
رابط مثل: https://salalah-rv-park.vercel.app
